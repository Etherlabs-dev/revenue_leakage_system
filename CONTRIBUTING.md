# Contributing

Thanks for your interest in improving the Revenue Leakage Detection System!

## Ways to contribute
- Fix bugs or edge cases in detection logic
- Add support for other billing platforms (Chargebee, Recurly, etc.)
- Improve docs, examples, or test data

## Workflow
1. Fork the repo
2. Create a feature branch: `git checkout -b feature/my-change`
3. Commit your changes with clear messages
